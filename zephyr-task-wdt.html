01 任务看门狗概览

任务看门狗（Task WDT）提供多级超时监控，支持软件和硬件看门狗链。

核心特性

特性说明
多级超时软件超时 → 硬件超时 → 系统复位
任务绑定每个看门狗通道绑定到特定任务
自动喂狗任务正常运行时自动喂狗
超时回调超时时可执行自定义处理

03 软件看门狗

// 任务看门狗
// CONFIG_TASK_WDT=y

// 安装看门狗通道
task_wdt_install(TASK_WDT_CHANNEL, timeout_ms, wdt_callback, NULL);

// 喂狗
task_wdt_feed(TASK_WDT_CHANNEL);

// 超时回调
static void wdt_callback(int channel_id, void *user_data)
{
    // 软件超时处理
    printk("任务看门狗超时!\n");
    // 可以：
    // 1. 记录错误日志
    // 2. 尝试恢复
    // 3. 触发硬件看门狗复位
}
