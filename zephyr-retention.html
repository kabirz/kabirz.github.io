01 Retention 概览

Retention 子系统提供跨重启的数据保留能力，用于传递状态和配置。

应用场景

场景说明
Bootloader → 应用传递启动原因、升级状态
崩溃日志保留崩溃信息供下次启动分析
OTA 状态保留固件升级进度
用户数据小量用户数据跨重启保留

02 保留区类型

类型说明
Flash 保留使用 Flash 分区保留数据
RAM 保留使用特殊 RAM 保留数据（需硬件支持）
寄存器保留使用 SoC 寄存器保留数据

03 Flash 保留

// Flash 保留区
// CONFIG_RETENTION=y

// 定义保留区（DTS）
// retention: retention@fd000 {
//     compatible = "zephyr,retention";
//     reg = <0xfd000 0x1000>;
// };

// 写入数据
const struct device *ret_dev = DEVICE_DT_GET(DT_NODELABEL(retention));
retention_write(ret_dev, 0, data, sizeof(data));

// 读取数据
uint8_t buf[64];
retention_read(ret_dev, 0, buf, sizeof(buf));
