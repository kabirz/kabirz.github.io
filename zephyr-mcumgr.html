01 mcumgr 概览

mcumgr（MCU Manager）是 Zephyr 的设备管理框架，提供 OTA 升级、OS 管理、统计收集等功能。

支持的命令组

命令组说明
Image (imgmgr)固件镜像管理（上传、测试、确认、回滚）
OS (os)OS 管理（复位、日期时间、任务列表）
Stats统计信息收集
Shell远程 Shell 访问
File System远程文件操作
Storage存储管理

03 SMP 协议

SMP（Simple Management Protocol）是 mcumgr 的传输协议，使用 CBOR 编码。

// SMP 协议格式
// +------+------+------+------+------+------+------+------+
// | len  | op   | flags| group| id   | seq  | data ...    |
// | 2B   | 1B   | 1B   | 2B   | 1B   | 1B   | variable    |
// +------+------+------+------+------+------+------+------+

// op 字段：
// 0x00 - request
// 0x01 - response
// 0x02 - notification

// group 字段：
// 0 - OS mgmt
// 1 - Image mgmt
// 2 - Storage mgmt
// 9 - Shell mgmt

04 镜像管理

// OTA 升级流程
// 1. 客户端上传新固件镜像
// 2. MCU 接收并写入 Flash
// 3. 客户端发送"测试"命令
// 4. MCU 重启，运行新固件
// 5. 应用发送"确认"命令
// 6. 新固件永久生效
// 7. 如果超时未确认，自动回滚

// 镜像状态
// IMAGE_STATE_PENDING    - 待测试
// IMAGE_STATE_CONFIRMED  - 已确认
// IMAGE_STATE_TESTING    - 测试中
// IMAGE_STATE_REVERTING  - 回滚中
// IMAGE_STATE_PENDING_CONFIRMED - 待确认

// Kconfig 配置
// CONFIG_IMG_MGMT=y
// CONFIG_IMG_MANAGER=y
// CONFIG_STREAM_FLASH=y

05 OS 管理

// OS 管理命令
