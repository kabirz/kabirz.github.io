01 网络管理概览

Zephyr 网络管理子系统提供网络接口管理、连接管理和事件通知。

管理组件

组件说明
net_mgmt网络管理事件框架
conn_mgr连接管理器
L2 管理链路层事件管理
接口管理网络接口控制

02 net_mgmt 框架

// net_mgmt 事件框架

// 注册事件回调
static void event_handler(struct mgmt_entry *entry, int status,
                          struct net_if *iface, void *data, size_t len)
{
    printk("网络事件: %d, 状态: %d\n", entry->cmd, status);
}

// 监听事件
struct mgmt_callback cb = { .handler = event_handler };
mgmt_register_event_callback(&cb, NET_REQUEST_IPV4_ADDR_ADD);

// 触发事件
mgmt_event_send(iface, NET_EVENT_IF_UP, NULL, 0);
