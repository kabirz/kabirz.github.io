01 概述

Swap（交换空间）是 Linux 的虚拟内存后备存储。当物理内存不足时，内核将不活跃的内存页面写入 swap 分区或 swap 文件，释放物理内存给活跃进程。Swap 是 overcommit 和 OOM 机制的重要补充。

02 Swap 类型

类型实现性能适用场景
Swap 分区专用磁盘分区较高服务器、生产环境
Swap 文件普通文件中等桌面、开发环境
zswap内存中的压缩缓存高减少磁盘 I/O
zram内存中的压缩块设备高嵌入式、无磁盘系统
swap over NFS网络交换低无盘工作站

03 换入换出流程

Swap Out（换出）流程：

  内存压力增大
        │
        ↓
  kswapd / direct reclaim 检查 LRU 链表
        │
        ├── 找到不活跃页面（inactive list）
        │
        ├── 检查页面是否可换出
        │   - 匿名页面（heap、stack、mmap）
        │   - 非脏页面（clean page cache）
        │
        ├── 分配 swap slot
        │   swap_info_struct → 找到空闲 swap 槽位
        │
        ├── 写入 swap 设备
        │   swap_writepage() → 提交 I/O
        │
        └── 更新页表
            PTE 标记为 swapped out
            记录 swap slot 信息

Swap In（换入）流程：

  进程访问被换出的页面
        │
        ↓
  缺页异常 → PTE 标记为 swapped out
        │
        ↓
  do_swap_page()
        │
        ├── 查找 swap slot
        │
        ├── 分配物理页面
        │
        ├── 读取 swap 设备
        │   swap_readpage() → 提交 I/O
        │
        ├── 更新页表
        │   PTE 指向新分配的物理页面
        │
        └── 数据恢复到物理内存

04 配置与调优

# 查看 swap 状态
$ free -h
$ swapon --show
$ cat /proc/swaps

# 创建 swap 文件
$ sudo fallocate -l 4G /swapfile
$ sudo chmod 600 /swapfile
$ sudo mkswap /swapfile
$ sudo swapon /swapfile

# 永久启用
$ echo '/swapfile none swap sw 0 0' >> /etc/fstab

# 调整 swappiness（0-100）
$ sysctl -w vm.swappiness=10   # 低值：尽量不使用 swap
$ sysctl -w vm.swappiness=60   # 默认值
$ sysctl -w vm.swappiness=100  # 高值：积极换出

# 查看 swap 使用情况
$ cat /proc/vmstat | grep pswpin/pswpout  # 换入/换出次数
$ vmstat -s | grep swap

Swappiness 指导

场景推荐 swappiness说明
数据库服务器1~10尽量避免 swap，保持 Page Cache
桌面系统60（默认）平衡内存和缓存
内存压力大80~100积极换出不活跃页面
Redis1与 vm.overcommit_memory=1 配合
