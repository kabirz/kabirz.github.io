01 概述

perf events 是 Linux 内核的性能分析框架，基于硬件性能计数器（PMU）提供 CPU 性能监控。它支持 CPU 周期计数、缓存命中率、分支预测、PMU 事件采样、软件事件计数等，是 Linux 性能分析的基石。

02 核心功能

功能命令说明
CPU 性能分析perf top实时显示 CPU 热点函数
采样分析perf record采样记录 → perf report 分析
统计计数perf statCPU 周期、指令数、缓存命中
火焰图perf script生成火焰图数据
硬件断点perf probe硬件断点监控
跟踪点perf trace系统调用跟踪
perf record 采样原理：

  ┌──────────────────────────────────────────────┐
  │                  CPU PMU                     │
  │                                              │
  │  硬件计数器：每 N 个 CPU 周期产生一次中断      │
  │  采样频率：99Hz / 999Hz / 任意频率            │
  │                                              │
  │  中断 → 记录当前 PC（程序计数器）              │
  │  → 记录到 ring buffer                        │
  │  → perf record 读取 ring buffer              │
  │  → 生成 perf.data 分析文件                   │
  └──────────────────────────────────────────────┘

  结果：统计每个函数被采样到的次数
  频率最高的函数 = CPU 热点（优化目标）

03 使用方式

# 性能统计
$ perf stat -p 1234 sleep 10
Performance counter stats for 'sleep 10':
  23456789  cycles
  12345678  instructions    # 0.53 insn per cycle
  2345678   cache-misses
  12345678  cache-references

# 采样分析
$ perf record -g -p 1234 -- sleep 30
$ perf report
# 查看热点函数

# 实时监控
$ sudo perf top
# 实时显示 CPU 热点

# 生成火焰图
$ perf script | stackcollapse-perf.pl | flamegraph.pl > flame.svg

# 系统调用跟踪
$ perf trace -p 1234 -s
# 类似 strace 但开销更低
