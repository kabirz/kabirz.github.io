01 Modbus 概览

Modbus 是工业自动化领域最常用的通信协议，Zephyr 支持 RTU 和 TCP 两种模式。

Modbus 特点

特性说明
主从架构客户端（主）发起请求，服务器（从）响应
RTU 模式串口通信（RS-232/485），二进制编码
TCP 模式以太网通信，MBAP 报文头
寄存器模型4 种寄存器类型（线圈、离散输入、保持、输入）

05 客户端

// Modbus 客户端（主站）

// 读取保持寄存器
uint16_t holding_regs[10];
modbus_read_holding_regs(ctx, slave_addr, start_addr, num_regs, holding_regs);

// 写入单个寄存器
modbus_write_single_reg(ctx, slave_addr, reg_addr, value);

// 写入多个寄存器
uint16_t values[] = {100, 200, 300};
modbus_write_multiple_regs(ctx, slave_addr, start_addr, num_regs, values);

// 读取线圈
uint8_t coils[8];
modbus_read_coils(ctx, slave_addr, start_addr, num_coils, coils);

// 写入线圈
modbus_write_single_coil(ctx, slave_addr, coil_addr, value);

06 服务器

// Modbus 服务器（从站）

// 寄存器定义
static uint16_t holding_regs[32];
static uint16_t input_regs[16];
static uint8_t coils[8];
static uint8_t discrete_inputs[8];

// 回调处理
static int holding_reg_cb(uint16_t addr, uint16_t *value, bool write)
{
    if (write) {
        holding_regs[addr] = *value;
    } else {
        *value = holding_regs[addr];
    }
    return 0;
}

// 注册回调
modbus_register_callbacks(ctx, &callbacks);

07 功能码

功能码名称说明
0x01读线圈读取离散输出（可写）
0x02读离散输入读取离散输入（只读）
0x03读保持寄存器读取可读写寄存器
0x04读输入寄存器读取只读寄存器
0x05写单个线圈设置单个线圈
0x06写单个寄存器设置单个保持寄存器
0x0F写多个线圈批量设置线圈
0x10写多个寄存器批量设置保持寄存器
