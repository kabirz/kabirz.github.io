01 RISC-V 架构

RISC-V 是开源指令集架构，FreeRTOS 支持 32/64 位处理器。

// RISC-V 特权级
// M 模式（Machine）— 最高特权（FreeRTOS 运行在此）
// S 模式（Supervisor）— 可选
// U 模式（User）— 最低特权

// 关键寄存器
// mtvec — 机器模式异常向量
// mepc  — 异常返回地址
// mcause — 异常原因
// mie   — 机器中断使能
// mip   — 机器中断挂起
// mstatus — 机器状态

02 中断处理

// RISC-V 中断处理流程
// 1. 中断/异常发生
// 2. CPU 自动保存上下文到 mepc, mcause
// 3. 跳转到 mtvec 指向的处理函数
// 4. 保存线程上下文到栈
// 5. 调用 C 处理函数
// 6. 恢复线程上下文
// 7. mret 返回

// 中断使能
// mie 寄存器：
// bit 3: MTIE（机器定时器中断使能）
// bit 7: MEIE（机器外部中断使能）
// bit 11: MSIE（机器软件中断使能）

03 PMP 保护

// PMP（Physical Memory Protection）
// 类似 ARM MPU，但更简单
// 8 个 PMP 条目

// PMP 配置寄存器
// pmpcfg0-3: 配置每个条目的权限
// pmpaddr0-7: 地址匹配

// PMP 标志
// R: 读
// W: 写
// X: 执行
// A: 地址匹配模式
// L: 锁定（不可修改）

// 地址匹配模式
// OFF: 禁用
// TOR: Top of Range（地址范围）
// NA4: Naturally Aligned 4-byte
// NAPOT: Naturally Aligned Power of Two
