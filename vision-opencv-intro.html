01 OpenCV 概述

OpenCV（Open Source Computer Vision Library）是一个开源的计算机视觉和机器学习软件库，由 Intel 于 1999 年发起，目前由 Willow Garage 和 Itseez（后被 Intel 收购）维护。它提供了超过 2500 个优化算法，涵盖图像处理、特征检测、目标跟踪、三维重建、机器学习等方向。

核心特性
  • 跨平台：支持 Linux、Windows、macOS、Android、iOS
  • 多语言接口：C++、Python、Java、MATLAB、JavaScript
  • 硬件加速：利用 SSE/AVX、CUDA、OpenCL、OpenVINO
  • 活跃社区：GitHub 上 80k+ star，每两个月发布小版本
  • 商业友好：BSD 协议，可自由使用和修改
典型应用场景
  • 人脸检测与识别
  • 自动驾驶车道检测与障碍物识别
  • 工业质检（缺陷检测、尺寸测量）
  • 增强现实（AR）标记点跟踪
  • 医学影像处理与分析
  • 安防监控中的运动检测与目标跟踪
版本说明:
OpenCV 4.x 是当前主流的稳定版本系列。相比 OpenCV 3.x 的主要变化包括：移除了 C API、强化了 DNN 模块、新增了 QR 码检测、改进了 ArUco 标记支持。目前 OpenCV 5.0 也在开发中，将引入更强大的 Transformer 支持和对新一代硬件的优化。

02 安装方式

2.1 pip 安装（Python 推荐）

对于 Python 开发者，pip 安装是最快捷的方式。OpenCV 官方发布四个 PyPI 包：

包名适用场景说明
opencv-python桌面开发包含核心模块和 imgproc/highgui 等，不包含 contrib 扩展
opencv-contrib-python需要扩展模块包含 opencv_contrib 扩展模块（SIFT、SURF、ArUco 等）
opencv-python-headless服务器/Docker无 GUI 组件，适合只做图像处理的服务器环境
opencv-contrib-python-headless服务器+扩展headless 版本 + contrib 扩展模块
# 基础安装（最常用）
pip install opencv-python

# 包含扩展模块（SIFT 等需要）
pip install opencv-contrib-python

# 指定版本
pip install opencv-python==4.10.0.84

# 使用清华镜像加速
pip install opencv-python -i https://pypi.tuna.tsinghua.edu.cn/simple

2.2 conda 安装

# conda 方式
conda install -c conda-forge opencv

# 创建新环境
conda create -n cv python=3.10
conda activate cv
conda install -c conda-forge opencv

2.3 源码编译（C++ 完整控制）

源码编译适合需要自定义编译选项（如 CUDA 支持、特定扩展模块）的场景。OpenCV 使用 CMake 构建系统。

# 1. 安装依赖（Ubuntu/Debian）
sudo apt update
sudo apt install build-essential cmake git pkg-config
sudo apt install libjpeg-dev libpng-dev libtiff-dev
sudo apt install libavcodec-dev libavformat-dev libswscale-dev
sudo apt install libgtk-3-dev libcanberra-gtk-module
sudo apt install libatlas-base-dev gfortran
sudo apt install python3-dev python3-numpy

# 2. 克隆源码
git clone https://github.com/opencv/opencv.git
git clone https://github.com/opencv/opencv_contrib.git

# 3. 创建构建目录
cd opencv && mkdir build && cd build

# 4. CMake 配置
cmake -D CMAKE_BUILD_TYPE=RELEASE \
      -D CMAKE_INSTALL_PREFIX=/usr/local \
      -D WITH_TBB=ON \
      -D WITH_V4L=ON \
      -D WITH_OPENGL=ON \
      -D WITH_CUDA=OFF \
      -D OPENCV_EXTRA_MODULES_PATH=../../opencv_contrib/modules \
      ..

# 5. 编译与安装
make -j$(nproc)
sudo make install
sudo ldconfig
编译优化建议:
如果需要 CUDA 加速，设置 -D WITH_CUDA=ON -D CUDA_ARCH_BIN=8.6（根据你的 GPU 架构调整）。对于嵌入式设备（如 Jetson），建议启用 -D WITH_NEON=ON。编译时长通常 15-60 分钟，取决于机器性能和启用的模块数量。

3.3 C++ 项目模板（CMake）

# CMakeLists.txt
cmake_minimum_required(VERSION 3.16)
project(VisionProject)

# 设置 C++ 标准
set(CMAKE_CXX_STANDARD 17)
set(CMAKE_CXX_STANDARD_REQUIRED ON)

# 寻找 OpenCV 包
find_package(OpenCV REQUIRED)
include_directories(${OpenCV_INCLUDE_DIRS})

# 可执行文件
add_executable(main main.cpp)
target_link_libraries(main ${OpenCV_LIBS})

将以上 CMakeLists.txt 放在项目根目录，执行 cmake . && make 即可构建。

04 第一个程序

4.1 Python 版本

import cv2
import numpy as np

# 读取图像
img = cv2.imread("example.jpg")
if img is None:
    print("图像读取失败，请检查文件路径")
    exit()

# 转换为灰度图
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

# 应用高斯滤波降噪
blurred = cv2.GaussianBlur(gray, (5, 5), 1.5)

# Canny 边缘检测
edges = cv2.Canny(blurred, 50, 150)

# 显示结果
cv2.imshow("Original", img)
cv2.imshow("Edges", edges)

# 等待按键后退出
cv2.waitKey(0)
cv2.destroyAllWindows()

# 保存结果
cv2.imwrite("edges_result.jpg", edges)
print("边缘检测结果已保存")

4.2 C++ 版本

#include <opencv2/opencv.hpp>
#include <iostream>

using namespace cv;
using namespace std;

int main(int argc, char** argv) {
    // 读取图像
    Mat img = imread("example.jpg");
    if (img.empty()) {
        cerr << "图像读取失败" << endl;
        return -1;
    }

    // 转换为灰度图
    Mat gray, blurred, edges;
    cvtColor(img, gray, COLOR_BGR2GRAY);

    // 高斯滤波
    GaussianBlur(gray, blurred, Size(5, 5), 1.5);

    // Canny 边缘检测
    Canny(blurred, edges, 50, 150);

    // 显示
    imshow("Original", img);
    imshow("Edges", edges);
    waitKey(0);
    destroyAllWindows();

    // 保存
    imwrite("edges_result.jpg", edges);
    cout << "边缘检测结果已保存" << endl;
    return 0;
}

4.3 相机实时处理

import cv2

# 打开默认相机（0）
cap = cv2.VideoCapture(0)
if not cap.isOpened():
    print("无法打开相机")
    exit()

while True:
    ret, frame = cap.read()
    if not ret:
        break

    # 水平翻转
    frame = cv2.flip(frame, 1)

    # 转换为灰度并画圆形标记
    gray = cv2.cvtColor(frame, cv2.COLOR_BGR2GRAY)

    # 显示帧率
    cv2.putText(frame, "Press ESC to exit",
                (10, 30), cv2.FONT_HERSHEY_SIMPLEX,
                0.8, (0, 255, 0), 2)

    cv2.imshow("Camera", frame)

    # ESC 键退出
    if cv2.waitKey(1) & 0xFF == 27:
        break

cap.release()
cv2.destroyAllWindows()
常见问题:
cv2.imshow() 在 headless 环境中不可用，此时应使用 opencv-python-headless 包，并通过 cv2.imwrite() 保存结果文件代替显示。如果出现 "libGL error: MESA-LOADER" 警告，安装 libgl1-mesa-glx 即可解决。

05 OpenCV 架构

5.1 Mat 数据结构

cv::Mat（n 维稠密数组）是 OpenCV 最核心的数据结构，用于存储图像、矩阵和任意多维数据。它本质上是一个引用计数的智能指针，管理一块连续（或非连续）的内存区域。

// Mat 内部结构示意
class Mat {
public:
    int dims;                 // 维度数（2 表示二维图像）
    int rows, cols;           // 行数和列数
    uchar* data;               // 指向像素数据的指针
    int type;                 // 数据类型（如 CV_8UC3）
    int channels();           // 通道数
    size_t step;               // 每行字节数（含填充）
    int refcount;             // 引用计数（共享数据）

    // 常用类型宏
    // CV_8UC1  — 8 位无符号单通道（灰度图）
    // CV_8UC3  — 8 位无符号三通道（BGR 彩色图）
    // CV_32FC1 — 32 位浮点单通道
    // CV_64FC3 — 64 位浮点三通道
};

Mat 的自动内存管理基于引用计数：

// Python 中 Mat 表现为 numpy.ndarray
img = cv2.imread("photo.jpg")
print(type(img))        # <class 'numpy.ndarray'>
print(img.shape)        # (480, 640, 3)
print(img.dtype)        # uint8

// C++ 中 Mat 自动管理内存
Mat a = imread("a.jpg");
Mat b = a;              // 共享数据，引用计数 +1
Mat c = a.clone();      // 深拷贝，独立内存
a.release();            // 引用计数 -1，数据区未释放

5.2 模块分层架构

OpenCV 采用分层模块化设计，核心层提供基础数据结构和算法，高层模块提供特定领域的应用功能。

├── core           基础数据结构（Mat、Point、Scalar、RNG）
├── imgproc        图像处理（滤波、形态学、直方图、几何变换）
├── highgui         GUI 与 I/O（imshow、窗口管理）
├── video          视频分析与运动估计（光流、背景减除）
├── videoio        视频编解码与相机接口
├── calib3d        相机标定与三维重建
├── features2d     特征检测与描述子（SIFT、ORB）
├── objdetect      目标检测（Haar Cascade、QR 码）
├── dnn            深度学习推理（ONNX、TensorFlow、Caffe）
├── ml             机器学习（SVM、K-Means、决策树）
├── flann          高维空间近似最近邻搜索
├── stitching      图像拼接
├── photo          计算摄影学（HDR、去噪、修图）
└── gapi           图形化管道加速（OpenCL/CUDA 后端）

扩展模块（opencv_contrib）包含更多实验性或与第三方库绑定的功能：

├── aruco          ArUco / AprilTag 标记检测   工业视觉定位
├── ximgproc       高级图像处理（SLIC 超像素、边缘检测）
├── xfeatures2d    实验性特征（SIFT 已移至此处）
├── tracking       目标跟踪（KCF、TLD、MIL）
├── face           人脸识别（LBPH、Fisherfaces）
├── text           OCR 文字检测与识别
├── bgsegm         改进的背景减除算法
└── wechat_qrcode  微信二维码引擎

06 常用模块详解

6.1 core 模块 — 基础与核心

core 模块提供所有其他模块依赖的基础设施：

组件说明
Matn 维稠密数组，图像和矩阵的存储容器
MatExpr矩阵运算表达式模板（延迟求值优化）
SparseMat稀疏矩阵（处理大部分元素为零的场景）
Point_ / Size_ / Rect_ / Scalar_基础几何类型
Vec / Matx固定尺寸小向量与矩阵（编译期优化）
RNG随机数生成器（Mersenne Twister）
UMat统一内存（OpenCL/CPU 透明切换）

6.2 imgproc 模块 — 图像处理

imgproc 是 OpenCV 中内容最丰富的模块，涵盖所有传统图像处理算法：

  • 色彩空间转换：cvtColor 支持 150+ 种颜色空间转换（BGR ↔ Gray / HSV / Lab / YCrCb / XYZ）
  • 几何变换：resize（缩放）、warpAffine（仿射）、warpPerspective（透视）、rotate（旋转）
  • 图像滤波：blur（均值）、GaussianBlur（高斯）、medianBlur（中值）、bilateralFilter（双边）
  • 形态学操作：erode（腐蚀）、dilate（膨胀）、morphologyEx（开/闭/梯度/顶帽/黑帽）
  • 边缘检测：Canny、Sobel、Scharr、Laplacian
  • 直方图：calcHist、equalizeHist、compareHist

6.3 highgui / videoio 模块 — I/O 与显示

highgui 提供图像显示和用户交互功能（imshow、namedWindow、滚动条），videoio 提供视频和相机接口：

import cv2

# highgui — 窗口管理
cv2.namedWindow("Preview", cv2.WINDOW_NORMAL)  # 可调整大小
cv2.setWindowTitle("Preview", "我的图像")
cv2.imshow("Preview", img)
cv2.resizeWindow("Preview", 800, 600)
cv2.waitKey(0)

# videoio — 读取视频文件
cap = cv2.VideoCapture("video.mp4")
fps = cap.get(cv2.CAP_PROP_FPS)
total_frames = cap.get(cv2.CAP_PROP_FRAME_COUNT)
print(f"帧率: {fps}, 总帧数: {total_frames}")

# 写入视频
fourcc = cv2.VideoWriter_fourcc(*'mp4v')
out = cv2.VideoWriter("output.mp4", fourcc, 30.0, (640, 480))
while cap.isOpened():
    ret, frame = cap.read()
    if not ret: break
    out.write(frame)

6.4 video 模块 — 视频分析

video 模块提供运动估计和目标跟踪算法：

  • 光流法：calcOpticalFlowPyrLK（稀疏 Lucas-Kanade）、calcOpticalFlowFarneback（稠密）
  • 背景减除：createBackgroundSubtractorMOG2、createBackgroundSubtractorKNN
  • 均值漂移：meanShift、CamShift（目标追踪）

6.5 features2d 模块 — 特征检测

features2d 提供了特征检测器与描述子的统一接口：

检测器关键点描述子许可

6.6 dnn 模块 — 深度学习推理

OpenCV 4.x 的 dnn 模块可以直接加载训练好的深度学习模型进行推理，无需安装 TensorFlow 或 PyTorch：

import cv2
import numpy as np

# 加载 ONNX 模型
net = cv2.dnn.readNetFromONNX("model.onnx")

# 预处理图像
blob = cv2.dnn.blobFromImage(img, scalefactor=1.0/255.0,
                             size=(224, 224), mean=(0, 0, 0),
                             swapRB=True, crop=False)

# 推理
net.setInput(blob)
output = net.forward()
dnn 模块支持的框架:
dnn 模块支持加载 Caffe（.caffemodel）、TensorFlow（.pb）、Torch（.pt）、Darknet（.weights）、ONNX（.onnx）、OpenVINO（.xml/.bin）等多种格式。推荐使用 ONNX 作为中间交换格式以获得最佳兼容性。
