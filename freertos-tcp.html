01 TCP 概览

FreeRTOS-Plus-TCP 是专为 FreeRTOS 设计的 TCP/IP 协议栈。

02 Socket API

Socket_t xSocket = FreeRTOS_socket(
    FREERTOS_AF_INET, FREERTOS_SOCK_STREAM, FREERTOS_IPPROTO_TCP);
FreeRTOS_bind(xSocket, &xAddress, sizeof(xAddress));
FreeRTOS_listen(xSocket, 20);
Socket_t xClient = FreeRTOS_accept(xSocket, NULL, NULL);
