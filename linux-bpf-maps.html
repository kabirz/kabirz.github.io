01 概述

BPF Map 是 eBPF 程序与用户态之间、以及 eBPF 程序之间的共享数据结构。它支持多种数据结构（哈希表、数组、环形缓冲区等），是 eBPF 实现有状态处理的核心机制。

02 Map 类型

类型结构用途
HASH哈希表通用键值存储
ARRAY固定大小数组计数器、统计
PERCPU_HASH每 CPU 哈希表无锁高并发统计
PERCPU_ARRAY每 CPU 数组无锁计数器
LRU_HASHLRU 哈希表自动淘汰旧条目
RINGBUF环形缓冲区高效事件流
STACK_TRACE栈追踪调用栈记录
DEVMAP设备映射XDP 重定向
CPUMAPCPU 映射XDP 跨 CPU 转发

03 API 与使用

// BPF 程序中使用 Map
struct {
    __uint(type, BPF_MAP_TYPE_HASH);
    __uint(max_entries, 1024);
    __type(key, __u32);
    __type(value, __u64);
} packet_count SEC(".maps");

SEC("xdp")
int xdp_counter(struct xdp_md *ctx) {
    __u32 key = 0;
    __u64 *val = bpf_map_lookup_elem(&packet_count, &key);
    if (val) __sync_fetch_and_add(val, 1);
    return XDP_PASS;
}

// 用户态读取 Map
int map_fd = bpf_map_create(...);
__u32 key = 0, next_key;
__u64 value;
bpf_map_lookup_elem(map_fd, &key, &value);
printf("packets: %llu\n", value);
