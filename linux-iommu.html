01 概述

IOMMU（I/O Memory Management Unit）是硬件设备和内存之间的DMA 重映射单元。它将设备的 IOVA（I/O 虚拟地址）翻译为物理地址，提供设备隔离、SVA（Shared Virtual Addressing）等能力。Intel VT-d 和 AMD-Vi 是两种主要实现。

02 架构

IOMMU 工作原理：

  无 IOMMU：
  设备 DMA → 直接访问物理内存 → 任何设备可访问任何内存

  有 IOMMU：
  设备 DMA → IOVA → IOMMU 翻译表 → 物理地址
  每个设备有独立的地址映射 → 设备只能访问分配给它的内存

IOMMU 三大功能：

  1. DMA 重映射
     设备的 DMA 地址翻译为物理地址
     防止恶意/故障设备访问未授权内存

  2. 设备直通（Passthrough）
     将物理设备直接分配给虚拟机
     虚拟机独占设备，性能接近原生

  3. SVA（Shared Virtual Addressing）
     设备和 CPU 共享同一虚拟地址空间
     设备可以直接使用 CPU 的页表
