01 FreeRTOS 概览

FreeRTOS 是全球最流行的嵌入式实时操作系统内核，由 Amazon Web Services 维护，支持 40+ 种处理器架构。

核心特性

特性说明
调度器抢占式 + 协作式 + 时间片轮转
任务数无限制（受 RAM 约束）
优先级0 ~ configMAX_PRIORITIES-1（数值越大优先级越高）
SMP 支持多核调度（FreeRTOS SMP）
MPU 支持内存保护单元集成
内存管理5 种方案（heap_1 ~ heap_5）
许可证MIT（商业免费）

02 内核架构

FreeRTOS 内核由 6 个核心模块组成，所有模块共用一个链表基础设施。

内核模块

模块源文件行数功能
任务管理tasks.c8872创建/删除/挂起/恢复/调度
队列queue.c~6000队列/信号量/互斥锁（统一实现）
定时器timers.c~2000软件定时器（守护任务实现）
事件组event_groups.c~1500位掩码事件等待
流缓冲stream_buffer.c~1500单生产者单消费者缓冲区
链表list.c~500双向链表（内核基础设施）

关键设计特点

FreeRTOS 设计哲学：
1. 极简内核 — 核心代码仅 ~9000 行 C
2. 统一队列 — 信号量/互斥锁/消息队列都基于 queue.c
3. 守护任务 — 定时器和流缓冲在专用任务中执行
4. 编译时配置 — 通过 FreeRTOSConfig.h 裁剪功能
5. 零动态分配 — 所有资源可静态分配

05 配置系统

FreeRTOS 通过 FreeRTOSConfig.h 进行编译时配置。

关键配置项

配置宏说明典型值
configUSE_PREEMPTION启用抢占式调度1
configCPU_CLOCK_HZCPU 时钟频率72000000
configTICK_RATE_HZ系统 tick 频率1000
configMAX_PRIORITIES最大优先级数56
configMINIMAL_STACK_SIZE最小栈大小（字）128
configTOTAL_HEAP_SIZE堆总大小（字节）32768
configUSE_TIMERS启用软件定时器1
configUSE_MUTEXES启用互斥锁1
configUSE_COUNTING_SEMAPHORES启用计数信号量1
configUSE_CO_ROUTINES启用协程0

08 快速上手

// FreeRTOS 最小配置示例

// FreeRTOSConfig.h
#define configUSE_PREEMPTION        1
#define configCPU_CLOCK_HZ         72000000
#define configTICK_RATE_HZ         1000
#define configMAX_PRIORITIES        5
#define configMINIMAL_STACK_SIZE    128
#define configTOTAL_HEAP_SIZE       10240

// main.c
#include "FreeRTOS.h"
#include "task.h"

void vTaskFunc(void *pvParameters) {
    while (1) {
        // 任务逻辑
        vTaskDelay(pdMS_TO_TICKS(1000));
    }
}

int main(void) {
    xTaskCreate(vTaskFunc, "Task1", 128, NULL, 1, NULL);
    vTaskStartScheduler();
    while (1) {}  // 不应到达这里
}
