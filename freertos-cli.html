01 CLI 概览

FreeRTOS-Plus-CLI 提供交互式命令行接口，支持命令注册和参数解析。

02 命令注册

static BaseType_t vCommand(char *pcWriteBuffer,
    size_t xWriteBufferLen, const char *pcCommandString) {
    strcpy(pcWriteBuffer, "Hello!\r\n");
    return pdFALSE;
}

static const CLI_CommandDefinition_t xCommand = {
    "hello", "hello: Print greeting\r\n",
    vCommand, -1
};
FreeRTOS_CLIRegisterCommand(&xCommand);
