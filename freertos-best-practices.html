04 最佳实践

1. 启用 configASSERT 验证配置
2. 使用静态分配避免堆问题
3. 启用栈溢出检测
4. 优先使用任务通知（最轻量）
5. ISR 中使用 FromISR API
6. 限制中断禁用时间
7. 使用 vTaskDelayUntil 实现周期任务
8. 优先使用互斥锁（非信号量）做同步
