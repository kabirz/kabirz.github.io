02 MPU / MMU 内存保护

MPU（Memory Protection Unit）和 MMU（Memory Management Unit）提供硬件级内存保护。

MPU（Cortex-M）

// MPU 配置
// CONFIG_HW_STACK_PROTECTION=y     ← 栈溢出检测
// CONFIG_USERSPACE=y               ← 用户态支持
// CONFIG_MPU_REQUIRES_POWER_OF_TWO_ALIGNMENT=y

// MPU 保护区域
// - 线程栈：读写，用户态可访问
// - 内核数据：读写，仅内核可访问
// - 外设寄存器：读写，仅驱动可访问
// - Flash：只读，所有代码可执行

// MPU 区域限制
// Cortex-M0+: 0 个区域（无 MPU）
// Cortex-M3/M4: 8 个区域
// Cortex-M7: 8 或 16 个区域
// Cortex-M33: 16 个区域

MMU（Cortex-A / x86）

// MMU 提供更灵活的内存保护
// CONFIG_MMU=y

// 虚拟内存映射
// - 页表保护
// - 按需分页
// - 写时复制（COW）
// - 虚拟地址空间隔离

03 用户态隔离

用户态线程通过系统调用访问内核服务，实现内核/用户态隔离。

// 启用用户态
// CONFIG_USERSPACE=y
// CONFIG_STACK_PROTECTION=y

// 用户态线程创建
K_THREAD_DEFINE(user_tid, STACK_SIZE,
                user_entry, NULL, NULL, NULL,
                K_PRIO_PREEMPT(5),
                K_USER,         // 用户态标志
                K_NO_WAIT);

// 系统调用机制
// 1. 用户态调用 API
// 2. 编译时生成的桩代码检查权限
// 3. SVC 中断切换到内核态
// 4. 内核执行实际操作
// 5. 返回用户态

// 权限控制
// - 线程只能访问自己内存域中的对象
// - 设备访问通过权限位图控制
// - 内核对象通过权限掩码保护

05 PSA 认证

PSA（Platform Security Architecture）提供标准化的安全 API。

// PSA Crypto API
// CONFIG_PSA_CRYPTO=y

// 密钥管理
psa_key_attributes_t attrs = PSA_KEY_ATTRIBUTES_INIT;
psa_set_key_type(&attrs, PSA_KEY_TYPE_AES);
psa_set_key_bits(&attrs, 128);
psa_set_key_usage_flags(&attrs, PSA_KEY_USAGE_ENCRYPT | PSA_KEY_USAGE_DECRYPT);

psa_key_handle_t key_handle;
psa_generate_key(&attrs, &key_handle);

// 加密
size_t output_len;
psa_cipher_encrypt(key_handle, PSA_ALG_CBC_PKCS7,
                   plaintext, plaintext_len,
                   ciphertext, ciphertext_size, &output_len);

// PSA 认证存储
// CONFIG_SECURE_STORAGE=y
// 使用 PSA API 安全存储密钥和数据

06 MCUboot 安全启动

MCUboot 提供安全启动、固件签名验证和回滚保护。

MCUboot 功能

功能说明
安全启动验证固件签名后再执行
固件升级支持 OTA 安全升级
回滚保护升级失败自动回退
加密存储加密固件镜像
// MCUboot 配置
// CONFIG_BOOTLOADER_MCUBOOT=y
// CONFIG_MCUBOOT_SIGNATURE=y          ← 签名验证
// CONFIG_MCUBOOT_ENCRYPTION=y         ← 加密固件
// CONFIG_IMG_MANAGER=y                ← 镜像管理
// CONFIG_STREAM_FLASH=y               ← 流式 Flash 写入

// 固件签名
// 使用 imgtool 生成签名密钥
// imgtool keygen -k signing-key.pem -t RSA-2048
// imgtool sign --key signing-key.pem firmware.bin signed.bin
